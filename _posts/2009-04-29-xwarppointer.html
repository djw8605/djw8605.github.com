--- title: XWarpPointer date: '2009-04-29T13:35:00.000-05:00' author: Derek Weitzel tags: - X modified_time: '2010-10-03T01:05:49.324-05:00' blogger_id: tag:blogger.com,1999:blog-3007054864987759910.post-6670607988879524138 blogger_orig_url: http://derekweitzel.blogspot.com/2009/04/xwarppointer.html --- Just came across a way to move the mouse pointer to an absolute position in X. Follow the code below:
int main()
{
        Display* xdisplay1 = XOpenDisplay(NULL);
        if(!xdisplay1) {
                printf("display = null\n");
        }
        Window root = DefaultRootWindow(xdisplay1);
        XWarpPointer(xdisplay1, 0, root, 0, 0, 0, 0, 10, 10);
        XCloseDisplay(xdisplay1);

}
